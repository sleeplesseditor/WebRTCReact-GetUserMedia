## Description
An initial introduction to the use of the [MediaStreams Web API](https://developer.mozilla.org/en-US/docs/Web/API/Media_Capture_and_Streams_API), with an updated version of the first coursework project for [Robert Bunch's Udemy course](https://www.udemy.com/course/mastering-webrtc-part-2-real-time-video-and-screen-share).

## Running the App Locally
- Run `npm install` in the root project folder
- CD into `client` folder
- Run `npm install`
- Return to root project folder and run `npm run dev`

## Changes from Original Project
- Revised JavaScript solution to Vite-based React project
- Updated script files to work as helper functions with use of React element Refs
